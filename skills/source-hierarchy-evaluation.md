# Source Hierarchy Evaluation

## When to use

Use this skill when the answer depends on whether a source should be trusted.

## SOP

1. Identify all source types involved.
2. Rank them by authority: primary source, official docs, raw data, reputable reporting, social posts, AI-generated replies.
3. Label X posts as primary statements, public discourse, or weak factual evidence depending on authorship and context.
4. Check whether the source directly supports the claim.
5. Prefer recent primary evidence for current facts.
6. State source limitations and avoid laundering weak evidence as authority.

## Evidence guidance

X access remains available. X should not be allowlisted by default, but its evidentiary role must be labeled clearly.
