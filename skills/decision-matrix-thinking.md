# Decision Matrix Thinking

## When to use

Use this skill when the user wants a recommendation between options.

## SOP

1. Identify options and decision criteria.
2. Weight criteria based on the user's constraints.
3. Score each option qualitatively or quantitatively.
4. Surface tradeoffs and dealbreakers.
5. Recommend the best option for the stated context.
6. Say what new information would change the recommendation.

## Evidence guidance

Use user constraints, technical requirements, cost, reliability, maintainability, and operational fit.

## Trace note

Show the criteria that drove the recommendation.
