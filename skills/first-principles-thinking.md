# First Principles Thinking

## When to use

Use this skill when the question requires understanding the underlying mechanism instead of repeating conventional explanations.

## SOP

1. Define the objective in one sentence.
2. List the hard constraints, known facts, and assumptions separately.
3. Reduce the problem to basic mechanisms or dependencies.
4. Rebuild the explanation or solution from those primitives.
5. Identify which assumptions would change the answer if they were false.
6. Present the answer with explicit constraints and caveats.

## Evidence guidance

Prefer deterministic constraints, source documents, system behavior, and direct measurements over social claims or loose analogies.

## Trace note

Show which assumptions and constraints shaped the answer. Do not show private chain-of-thought.
